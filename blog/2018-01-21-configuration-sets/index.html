很实用的一些配置集合

January 21, 2018

换设备经常会用到的一些东西、有些比较容易忘，在此记录一下

Vim #

显示行号 #

  1. 永久
echo 'set number' >> ~/.vimrc
  1. 临时
: set number

批量注释&&取消注释 # 

:起始行号,结束行号s/^/注释符/g
:起始行号,结束行号s/^注释符//g

批量替换 # 

:%s/源字符串/目的字符串/g

tab设置为4个空格 # 

set ts=4
set expandtab

Shell命令 #

scp #

  1. 上传
scp 本地文件完整路径 username@ip:远程目录路径
  1. 下载
scp username@ip:远程文件完整路径 本地目录

VSCode #

更换默认shell # 

https://code.visualstudio.com/docs/editor/integrated-terminal#_configuration

列编辑 #

alt + shift + 鼠标滚动

OS #

Mac环境变量更改 # 

cd /etc/paths.d
touch go //go环境变量
touch nginx //nginx环境变量

参考: mac设置环境变量

VCS #

Git免密Pull&Push # 

git config --global credential.helper store --file ~/.my-credentials

参考：Git工具-凭证存储
